name: docs-update
description: Met à jour la documentation en cohérence avec le code actuel
---

Met à jour la documentation du projet via l'agent `docs-update`.

Utilise l'outil Agent avec `subagent_type: docs-update` pour analyser les changements en cours et corriger les sections documentaires devenues inexactes.

Périmètre ciblé (optionnel) : $ARGUMENTS

Si aucun argument, analyser toute la documentation.
